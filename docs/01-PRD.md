# Product Requirements Document

## Funcionalidades Principais

1. Consultar clientes sem atendimento no dia anterior.
2. Consultar histórico real de pedidos dos clientes encontrados.
3. Disparar mensagem automática de saudação via z-api (WhatsApp).
4. Receber respostas dos clientes via webhook.
5. Validar identidade via CPF/CNPJ.
6. Direcionar atendimento para telemarketing ou sugestão de pedido.
7. Permitir edição dinâmica da sugestão de pedido.
8. Encerrar sessões automaticamente por timeout ou erro.
